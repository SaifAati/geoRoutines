# Installation
To install `geoRoutines`from sources:

    git clone https://github.com/SaifAati/geoRoutines.git
    cd geoRoutines
    conda env create -- file geoCosiCorr3D.yml
    conda activate geoCosiCorr3D
    pip install -e .
